Redux

Store
Redux 提供 createStore 函数来创建 Store 对象
整个应用只能有一个 Store 对象
```
import { createStore } from 'redux';
const store = createStore(rootReducer);
```

Action
```
const action = {
    type: 'entry/login',
    payload: {username: 'username', password: '123456'}
}

store.dispatch(action);
```

Reducer
Reducer 作用就是收到 Action, 计算新的 state
```
const reducer = (state, action) => {
    switch(action.type) {
        case 'entry/login':
            handleLogin();
        default:
            return state;
    }
}
```

React Redux 

安装
```
npm install react-redux --save
```

React Redux 提供 <Provider /> 组件, 使整个应用可以使用 Store
```
import { Provider } from 'react-redux'

ReactDOM.render((
  <Provider store={store}>
    <App />
	</Provider>
), document.getElementById('app'));
```

```
npm install --save redux-observable
```

Epic

Epic是一个函数，参数是动作流合状态流，返回动作流

```
import { ofType } from 'redux-observable';

const loginEpic = (action$, state$) => action$.pipe(
  ofType('entry/login'),
  delay(1000),
  mapTo({ type: 'PONG' }),
)
```
