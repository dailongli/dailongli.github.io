Cocos2d-x是一个C++的开源跨平台游戏引擎。

macOS下环境搭建
```
$ wget http://digitalocean.cocos2d-x.org/Cocos2D-X/cocos2d-x-3.17.zip
$ unzip cocos2d-x-3.17.zip
$ cd cocos2d-x-3.17

$ cocos gen-libs -c -p ios

$ ./setup.py
$ source ~/.bash_profile

$ cocos -v
```




创建新lua项目
```
$ cocos new MyGame -p com.your_company.mygame -l lua -d ~/Projects/
$ cd ~/Projects/MyGame

删除
libsimulator.xcodeproj
cocos2d_lua_bindings.xcodeproj
cocos2d_libs.xcodeproj


$ cocos gen-libs -c -p ios
$ rm frameworks/cocos2d-x

Build Phases --> Link Binary With Libraries 添加 
~/cocos2d-x-3.10/prebuild/ios/libcocos2d iOS.a
~/cocos2d-x-3.10/prebuild/ios/libluacocos2d iOS.a
~/cocos2d-x-3.10/prebuild/ios/libsimulator iOS.a
```

Build Settings ---> Header Search Paths
add cocos header



安装android-ndk
```
$ brew tap homebrew/cask
$ brew cask install android-sdk android-ndk

export NDK_ROOT=/usr/local/share/android-ndk
export PATH=$NDK_ROOT:$PATH

$ source ~/.bash_profile
```
