Cocos2d是一个Python开源游戏框架，Cocos2d-x是用C++写的。



macOS下环境搭建
```
$ git clone https://github.com/cocos2d/cocos2d-x.git
$ cd cocos2d-x
$ ./download-deps.py
$ git submodule update --init

$ open ./build/cocos2d_tests.xcodeproj
选择 "cpp-tests Mac", build and run
```

安装android-ndk
```
$ brew tap homebrew/cask
$ brew cask install android-sdk android-ndk

export NDK_ROOT=/usr/local/share/android-ndk
export PATH=$NDK_ROOT:$PATH

$ source ~/.bash_profile
```

```
$ ./setup.py
$ source ~/.bash_profile
```

创建新lua项目
```
$ cocos new MyGame -p com.your_company.mygame -l lua -d ~/Projects/
$ cd ~/Projects/MyGame
```
